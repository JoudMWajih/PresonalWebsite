# Personal Website

## Introduction

This project is a simple responsive personal website built using HTML, CSS, and JavaScript. The website presents personal information, skills, projects, and contact details in a clear and organized layout.

## Website Idea

The website is designed as a personal portfolio page that introduces who I am, my academic background, technical interests, skills, and selected projects. The focus is on creating a clean and user-friendly interface.

## Technologies Used

- HTML
- CSS
- JavaScript

## Website Sections

- Home
- About Me
- Skills
- Projects
- Contact

## Features

- Simple and clean design
- Responsive layout for different screen sizes
- Organized content sections
- Basic interaction using JavaScript
